## Install

```bash
npm install -g i18next-to-xlsx
```

## Usage

```text
Usage: i18next-to-xlsx [--import]
```

## Examples

`From locale directory:`

export:
```bash
> i18next-to-xlsx
```
import:
```bash
> i18next-to-xlsx --import
```
